Connecting to Register Server ... 
PowerCam 每一套序号可以上网注册两次，并提供两种注册模式，说明如下:

[Internet]
若您的电脑可以连上网际网路 (Internet)，请输入软体序号后点选下方 Internet 按钮，即可完成注册并传回相关资讯 (如公司名称、剩余次数)。

[Verified Code]
当您的电脑无法连上网路注册，请依下列步骤注册:
    1. 将检查码纪录
    2. 找可以上网的电脑，连上 http://register.powercam.com.tw/getkey.php
    3. 输入序号、检查码，确定后会得到ㄧ个认证码。
    4. 点选 Verified Code 按钮，输入认证码即可完成注册。

若有任何的疑问，请与台湾数位学习科技联系 (http://www.powercam.com.tw)。
